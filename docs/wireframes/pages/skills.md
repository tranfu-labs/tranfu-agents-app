## 注释

| 编号 | 元素 | 状态/交互 | 数据来源 | 引用控件 |
|---|---|---|---|---|
| ① | 视角切换卡片（按 Skill / 按人） | 独立 `frame`；标题栏 `cnt` 随视角显示说明；内容行为 32px 分段按钮；切视角不重置时间窗；按人视角隐藏 ④ 内 Lens | URL query `view` | ViewSwitch |
| ② | SKILLS 统计 + 过滤条（搜索/runtime/来源/时间窗） | 改条件即筛选；写入 URL query；搜索 placeholder 随 ① 切换；手机单列，平板可换行；标题右侧承载 ⑥ loading/error 文案 | `/api/skills?days={7|30|90}` + URL query `win/rt/src/q/sort/dir/view/lens` | FilterBar |
| ③ | 每日使用 Top8 堆叠柱状图 | 按当前视角用 skill/operator 分段；悬停或点击当天柱显示锚定浮窗；7d 手机铺满容器，30/90d 只在图表内横滚；全空时显示 Empty | overview.daily / overview.operator_daily（only used） | StackedSkillChart |
| ④ | 使用排行主榜（只排 used）+ 管理者 Lens | 三态：`view=skill,lens=all` 显示 Skill 主榜；`view=skill,lens=untracked` 显示未收录使用占比榜；`view=operator` 显示按人榜且隐藏 Lens。手机三态都压缩为摘要行；整行可点 → `/skill/:name` 或 `/operator/:name`；Enter/Space 可达；表头排序不触发行跳转 | overview.table / overview.operator_table / overview.governance.untracked_usage | SkillsTable · OperatorTable · GovernanceLens · GovernanceSkillTable |
| ⑤ | 公司库采纳漏斗：采集/已装/近30天用/闲置（own+meta） | 展开看清单；显示 installed snapshot 说明；catalog stale 时显示警告；从未成功拉取时显示 `catalogUnavailable` 空态；平板/手机位于使用排行之后 | overview.funnel / overview.catalog | Funnel · Empty · details |
| ⑥ | 加载 / 错误 / 空态 | 首次加载只保留 ① + Loading frame；后续刷新 loading/error 写入 ② 标题右侧 `cnt`；Skill/未收录/按人空榜分别显示对应 Empty；图表全空不画空轴 | useSkillsOverview loading/error + 各表过滤结果 | Empty · SectionTitle `cnt` |
| ⑦ | 三态主题控件 | `system` / `light` / `dark` 三态；当前模式选中；键盘可达；刷新后保留显式选择；`system` 跟随系统偏好 | `tf-theme-mode` + `prefers-color-scheme` | SegmentedControl |
