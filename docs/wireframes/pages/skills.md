## 注释

| 编号 | 元素 | 状态/交互 | 数据来源 | 引用控件 |
|---|---|---|---|---|
| ① | 标题 + 过滤条（搜索/runtime/来源/时间窗） | 改条件即筛选；写入 URL query | `/api/skills?days={7|30|90}` | FilterBar · input · select |
| ② | 每日使用 Top8 堆叠柱状图 | 悬停看当天分项 | overview.daily（only used） | StackedSkillChart |
| ③ | 使用排行主榜（只排 used） | skill 名可点 → `/skill/:name`；表头排序 | overview.table | SkillsTable · RuntimeBars · MiniTrend |
| ④ | 公司库采纳漏斗：采集/已装/近30天用/闲置（own+meta） | 展开看清单 | overview.funnel | Funnel · details |
