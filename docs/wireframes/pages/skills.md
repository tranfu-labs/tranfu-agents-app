## 注释

| 编号 | 元素 | 状态/交互 | 数据来源 | 引用控件 |
|---|---|---|---|---|
| ① | 视角切换卡片（按 Skill / 按人） | 独立 `frame`；标题栏 `cnt` 随视角显示说明；内容行为 32px 分段按钮；切视角不重置时间窗 | URL query `view` | ViewSwitch |
| ② | SKILLS 统计 + 过滤条（搜索/runtime/来源/时间窗） | 改条件即筛选；写入 URL query | `/api/skills?days={7|30|90}` | FilterBar |
| ③ | 每日使用 Top8 堆叠柱状图 | 按当前视角用 skill/operator 分段；悬停看当天分项 | overview.daily / overview.operator_daily（only used） | StackedSkillChart |
| ④ | 使用排行主榜（只排 used） | 整行可点 → `/skill/:name` 或 `/operator/:name`；Enter/Space 可达；表头排序不触发行跳转 | overview.table / overview.operator_table | SkillsTable · OperatorTable |
| ⑤ | 公司库采纳漏斗：采集/已装/近30天用/闲置（own+meta） | 展开看清单 | overview.funnel | Funnel · details |
