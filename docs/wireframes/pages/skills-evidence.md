## 注释

| 编号 | 元素 | 状态 / 交互 | 数据来源 | 引用控件 |
|---|---|---|---|---|
| ① | 紧凑页头 | 返回 `/skills` 时删除 evidence-only 参数并保留原窗口和筛选；展示实际生效筛选与 ignored filters；`kind=total` 的未收录数量作为句内切片跳治理 clue | URL query + `/api/skills/evidence` | SkillsEvidenceView |
| ② | 原始记录主表 | 有 raw records 的 kind 默认停在原始记录；1440×900 第一屏露出表头和前几行；超过首批时通过原生 button 加载更多，展示已加载/总量；Tab 聚焦，Enter/Space 触发；错误或超时时保留已加载记录并允许重试；筛选 URL 变化时丢弃旧响应；到底后显示完成态；无下钻目标，不呈现可点态；时间按浏览器本地时区展示 | `summary.records` + `records` + `limit/offset` | records-table / evidence-load-more |
| ③ | Top skills | 辅助分组，排在主表之后；不足以并排时下置，不压窄主表 | `top_skills` | evidence-list |
| ④ | Top operators | 辅助分组，空 operator 不进入 Top；排在主表之后 | `top_operators` | evidence-list |
| ⑤ | 名单主表 | 仅名单型 kind 兼容展示；治理线索首选 `pages/skills-clues.md`；最近使用日保持服务端 `day` 语义 | `items` | mobile-card-table |
