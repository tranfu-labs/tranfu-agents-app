## 注释

| 编号 | 元素 | 状态/交互 | 数据来源 | 引用控件 |
|---|---|---|---|---|
| ① | 区块标题 + 会话总数 | 静态展示 | `/api/state` sessions 长度 | SectionTitle |
| ② | 会话表格行（Agent/调度员/运行时/模型/状态/今日/Skill/MCP/Shim/成功率） | Agent 名可点 → `/agent/:key`；表头点击排序 | `/api/state` sessions[] | table · Link · QBar · ShimPill |
| ③ | 三态主题控件 | `system` / `light` / `dark` 三态；当前模式选中；键盘可达；刷新后保留显式选择；`system` 跟随系统偏好 | `tf-theme-mode` + `prefers-color-scheme` | SegmentedControl |
