## 注释

| 编号 | 元素 | 状态/交互 | 数据来源 |
|---|---|---|---|
| ① | 顶部控制条 | Runtime/操作员视角切换与 `rank/q/status/w/wstart/wend/rt/op/sort` 写入 URL；移动端默认折叠；变化 replace | 当前 `/agents` URL + `/api/state.sessions` |
| ② | 时间窗变化 | 四个独立卡片展示当前/上一同长度窗口的活跃 Agent、活跃时长变化，以及当前在线和运行质量快照；缺失前期显示 `—` | `/api/state.agent_overview` + 窗口纯函数 |
| ③ | 摘要事实 | 保留 Agent 总数、运行中、今日/本周活跃、运行质量、待处理 Agent | `/api/state.agent_overview.summary` |
| ④ | 问题线索 | 与 Skills 健康条一致的紧凑事实项；点击写入 `status=attention&signal=...` | `sessions[]` + Shim 三态 + 纯函数分类 |
| ⑤ | 主分析区 | 桌面左排行、右趋势且底边对齐；`≤1080px` 单列；长轴只在图表盒子内横滚 | 当前时间窗聚合结果 |
| ⑥ | Agent 卡片 | 整卡链接与键盘 Enter/Space 下钻 `/agent/:key`；展示任务、步骤、活跃、Skill、MCP、质量、Shim | `/api/state.sessions[]` |
| ⑦ | 主题/语言 | 复用全局三态主题、中英文切换；Agents 新增状态不持久化 | TopBar + `tf-theme-mode` 主题例外 |
