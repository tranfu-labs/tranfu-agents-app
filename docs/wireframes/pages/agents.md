## 注释

| 编号 | 元素 | 状态 / 交互 | 数据来源 | 引用控件 |
|---|---|---|---|---|
| ① | 控制条 / 手机摘要 | 桌面平板标题为“控制条”并显示当前视角说明；切换顺序为“操作员｜运行终端”，默认操作员；运行终端显式写 `rank=runtime`；手机默认折叠 | `/agents` URL | 当前 Agents toolbar，视觉参照 Skills toolbar |
| ② | 今天变化 KPI 网格 | 八张同构卡；数值与真实入口同行；前两张显示当前/上期与 delta，其余六张显示快照和 detail；桌面 8×1、平板 4×2、手机 2×4 | `agent_overview` + window helper | Skills KPI grid 视觉语法 + AgentKpiGrid 数据 |
| ③ | 问题线索 | 点击回填 `status=attention&signal=...`；手机优先显示 | sessions + shim + quality | 当前 agents health |
| ④ | 排行 | 操作员/运行终端视角在控制条；点击回填 `op/rt`；跨行列轨道稳定，所有进度条同起点 | window overview groups | AgentRankPanel |
| ⑤ | 单日分布 / 多日趋势 | 单日正值按当前操作员/运行终端视角与指标显示环形扇形图，中心显示总值；多日仍逐日堆叠；共同使用 Top 8 + 其他、颜色、图例、tooltip、hover/click/focus 与今日斜纹；零值 Empty、长窗图内滚；不改 Skills | filtered sessions `active_days` + window days | AgentActivityChart + `resolveSkillsChartLayout` |
| ⑥ | Agent 明细 | 与控制条筛选和时间窗使用同一可见集合；窗口时长/活跃天数随 `w` 重算，质量明确为累计；桌面完整表格、平板盒内横滚、手机摘要行；整行键盘可达下钻 | filtered sessions `active_days` | AgentDirectoryTable |
