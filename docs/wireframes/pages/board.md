## 注释

| 编号 | 元素 | 状态/交互 | 数据来源 | 引用控件 |
|---|---|---|---|---|
| ① | 区块标题 Mission Pods + 计数/在线数 | 静态展示 | `/api/state` totals.live、sessions 长度 | SectionTitle |
| ② | Pod（按 operator 分组）头 + Agent 卡片网格 | 卡片整块可点 → `/agent/:key` | `/api/state` sessions 按 operator 归并 | Link 卡片 |
| ③ | 活动流 feed，最近事件倒序 | 静态列表 | `/api/state` feed[] | Feed 列表 |
| ④ | 三态主题控件 | `system` / `light` / `dark` 三态；当前模式选中；键盘可达；刷新后保留显式选择；`system` 跟随系统偏好 | `tf-theme-mode` + `prefers-color-scheme` | SegmentedControl |
