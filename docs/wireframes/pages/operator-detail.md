## 注释

| 编号 | 元素 | 状态/交互 | 数据来源 | 引用控件 |
|---|---|---|---|---|
| ① | 头部：操作员名 | 返回 `/skills?view=operator...` | `/api/operator/{name}` operator | dhead |
| ② | 指标卡：近7d/近30d/总计/用过 skill/会话/首次/最近 | 静态 | detail.metrics | statgrid |
| ③ | 口径说明 | 常驻 | 固定文案 | note-warn |
| ④ | 日级趋势 | 按 skill 分段 Top8+其它，今日进行中 | detail.daily | StackedSkillChart |
| ⑤ | runtime 分布 | 左窄栏；静态 | detail.runtime | Distribution |
| ⑥ | 使用 Skill 排行 | 右宽栏；默认按 7 天降序；整行可点 → `/skill/:name`，Enter/Space 可达 | detail.skills | table |
| ⑦ | 最近记录 | 最近 50 条，时间列取 `first_seen` 按浏览器本地时区展示；本地今天内显示相对时间，昨天及更早显示绝对时间，hover 显示本地绝对时间+时区；缺失 `first_seen` 回退原始日期；mode 恒 used 故省略；不可点 | detail.records[] | table |
