## 注释

| 编号 | 元素 | 状态 / 交互 | 数据来源 | 引用控件 |
|---|---|---|---|---|
| ① | clue 页头 | 返回 `/skills` 时删除 clue-only 参数并保留原窗口和筛选；chip 显示 `近 7 天`、日期范围、`来源：未收录/公司库`、skill/operator/runtime，不显示 raw key 或 raw enum | URL query + `applied_filters` | SkillsClueView |
| ② | Top Operators / 名单首屏 | `untracked` 首屏先露 Top Operators，显示 `records/total · percent`；`idle` 首屏露安装者名单；`zero-install` 首屏露零装机名单 | `/api/skills/evidence` | SkillsClueView |
| ③ | 未收录 records | 仅 `untracked` 展示；带 `skill=` 时隐藏 Top Skills，records 表不重复 Source 列 | `records/top_operators/top_skills` | records-table |
