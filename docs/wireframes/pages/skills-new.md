## 注释

| 编号 | 元素 | 状态/交互 | 数据来源 | 引用控件 |
|---|---|---|---|---|
| ① | 页头 | 返回 `/skills`，保留进入时的窗口和搜索 query；显示当前窗口和 catalog stale/unavailable 提示 | `/api/skills.window`、`catalog`、`period_comparison.current_published_skill_count` | Link · StatusLine |
| ② | 控制条 | 支持窗口、custom 起止、skill 搜索和来源 `全部/own/meta`；筛选变化写 URL query 并 replace | `useSkillQueryState` + `/api/skills` | Toolbar · Select · Input |
| ③ | 新发布列表 | 展示当前窗口 `published_skills[]`；有 used 详情行可 Enter/Space 跳 `/skill/:name`，未 used 行保持只读 | `/api/skills.published_skills[]` | Table · MobileSummaryRow |
