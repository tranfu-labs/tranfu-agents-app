# ADR-0001 单容器 + SQLite,无外部依赖

- 状态:Accepted
## 背景
团队工具要让任何人都能 5 分钟自托管,部署越简单越好;数据量小(极简遥测)。
## 决策
服务端是**单个 FastAPI 进程**,既收事件又发看板与 API;数据落**单文件 SQLite**(`$TF_DB`)。
部署 = 一个容器(根目录 `compose.yml`,挂卷 `tf-data`),不依赖外部数据库/缓存/MQ。
## 后果
- ✅ 512MB 机器即可;备份=拷 `tf.db`;无运维负担。
- ⚠️ 单写者并发上限有限(团队级遥测足够);如需横向扩展才考虑换库——届时另起 ADR。
- 约束:禁止为看板引入外部 DB/MQ 或独立前端构建。
