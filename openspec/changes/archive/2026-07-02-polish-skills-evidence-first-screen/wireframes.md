## flow.md 目标补充

```text
┌─ 页面 /skills ─────────────┐ ① icon evidence     ┌─ 页面 /skills/evidence ───────┐
│摘要格 / 问题线索 / 待办行 │────────────────────▶│默认原始记录或名单主表          │
│mobile 行点击 evidence      │                     │未收录切片可跳 kind=untracked  │
└────────────────────────────┘◀──── ② 返回 ───────└────────────────────────────────┘
```

| 步 | 从 | 到 | 触发 |
|---|---|---|---|
| ① | `/skills` 摘要格、问题线索、待处理线索事实行 | `/skills/evidence?kind=...` | 点 icon evidence；mobile 待办行点击；保留当前窗口和筛选 query |
| ② | `/skills/evidence` | `/skills` | 点 `← SKILLS`，保留原窗口和筛选 query |
