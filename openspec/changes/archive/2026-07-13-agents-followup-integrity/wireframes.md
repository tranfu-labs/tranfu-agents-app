# Agents 运营看板 · agents-followup-integrity

> 基线：`docs/wireframes/pages/agents.md`。本 change 收尾摘要事实、custom 时间窗和排行榜视角 URL 状态。

```text
━━ 桌面 1440×900 ━━
┌─ // Agents 运营看板 ───────────────────────────── 共 N 个 · M 个运行中 ┐
│ [Runtime | 操作员] [搜索…] [状态] [时间窗: 今天] [Runtime] [操作员]  │
├─ 时间窗变化 · 今天 ────────────────────────────────────────────────┤
│ [活跃 Agent 卡] [活跃时长卡] [当前在线卡] [运行质量卡]              │
├─ 摘要 ───────────────────────────────────────────────────────────────┤
│ [Agent 总数] [运行中] [今日/本周活跃] [运行质量] [待处理 Agent]       │
├─ 问题线索（Skills 紧凑事实条） ──────────────────────────────────────┤
├─ Runtime / 操作员排行 ───────────────┬─ 时间窗活跃趋势 ──────────────┤
│                                      │                              │
├─ Agent 明细 ─────────────────────────────────────────────────────────┤
```

custom 输入只在控制条展开时出现；开始/结束时间逐个填写也保留 URL。排行榜视角切换写入 `rank`，移动端仍默认折叠控制条，根页面不得横向滚动。
