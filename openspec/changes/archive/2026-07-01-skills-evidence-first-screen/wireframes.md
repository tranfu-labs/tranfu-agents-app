## flow.md 目标

```text
┌─ SKILLS 统计 /skills ─┐  ① 证据入口       ┌─ SKILLS 证据页 /skills/evidence ─┐
│ → pages/skills.md     │ ───────────────▶ │ → pages/skills-evidence.md       │
│                       │ ◀── ② ←SKILLS ─ │                                  │
└───────────────────────┘                  └──────────────────────────────────┘
```

| 步 | 从 | 到 | 触发 |
|---|---|---|---|
| ① | `/skills` 首屏摘要格、问题线索、待处理线索、排行行 | `/skills/evidence?kind=...` | 点 `看证据` / `看名单`，保留当前 window 和筛选 query |
| ② | `/skills/evidence` | `/skills` | 点 `← SKILLS`，保留原 window 和筛选 query |
