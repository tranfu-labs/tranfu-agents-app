## 注释

| 编号 | 元素 | 状态/交互 | 数据来源 | 引用控件 |
|---|---|---|---|---|
| ① | 页面头部或视角卡 | `/skills` 视角切换写 URL query;详情页返回 SKILLS | URL query / route param | frame / dhead |
| ② | 筛选条或指标卡 | 手机单列筛选;详情指标卡手机 2 列 | `/api/skills` / detail metrics | FilterBar / statgrid |
| ③ | 趋势图 | 图表内部横滚;悬停/点击浮窗视口避让 | daily / detail.daily | StackedSkillChart / DetailTrend |
| ④ | 使用排行或 runtime 分布 | `/skills` 排行手机摘要行且可下钻;详情分布单列 | overview.table / runtime | table / Distribution |
| ⑤ | 公司库漏斗或 operator/skill 分布 | `/skills` 漏斗跟在排行后;详情分布单列 | funnel / operators / skills | Funnel / Distribution / table |
| ⑥ | 最近记录 | 手机摘要行;无下钻目标,不可点击 | detail.records[] | records-table |
