# specs/board delta：最近记录采用通用日期展示

## 修改

- 操作员详情与单 skill 详情的"最近记录"列表中,若 `first_seen` 有效且不晚于当前时间:
  - 落在浏览器本地今天内时,可见文本保持克制相对时间:
    中文 `刚刚` / `N分钟前` / `N小时前`,英文 `just now` / `Nm ago` / `Nh ago`。
  - 落在浏览器本地昨天时,可见文本为中文 `昨天 HH:mm`,英文 `yesterday HH:mm`。
  - 落在浏览器本地 2-6 天前时,可见文本为星期标签加本地时刻:
    中文 `周一 HH:mm` 等,英文 `Mon HH:mm` 等。
  - 落在浏览器本地今年更早时,可见文本为 `MM-DD HH:mm`。
  - 落在浏览器本地非今年时,可见文本为 `YYYY-MM-DD HH:mm`。
  - 鼠标悬浮时间单元格时,title 仍显示完整本地绝对时间与浏览器时区名到秒
    (如 `2026-06-27 23:00:00 Asia/Shanghai`;时区名不可得时省略)。

## 不变

- `first_seen` 晚于当前时间时仍显示完整本地绝对时间,不得显示负相对时间。
- `first_seen` 缺失时,不得把 date-only `day` 强行按浏览器时区换算或补造具体时刻;只能按服务端返回的
  `Asia/Shanghai` `today` 与原始 `day` 显示相对/日历日期:
  中文 `今天` / `昨天` / `周一` / `MM-DD` / `YYYY-MM-DD`,英文 `today` / `yesterday` / `Mon` /
  `MM-DD` / `YYYY-MM-DD`;hover title 保留原始 `day`。
- `day`、`first_day`、`last_day`、`today`、SKILLS 图表日轴与日级聚合仍是服务端 `Asia/Shanghai` 统计日口径。

## 可验证行为

- 浏览器时区为 Asia/Shanghai,当前本地时间为 `2026-06-28 01:00:00`,记录
  `first_seen=2026-06-27T16:30:00+00:00` -> 最近记录可见文本为 `30分钟前`,hover title 为
  `2026-06-28 00:30:00 Asia/Shanghai`。
- 同一当前时间,记录 `first_seen=2026-06-27T15:00:00+00:00` -> 最近记录可见文本为
  `昨天 23:00`,hover title 为 `2026-06-27 23:00:00 Asia/Shanghai`。
- 当前本地时间为 `2026-06-30 12:00:00`,记录本地时间为 `2026-06-25 09:18:55` ->
  最近记录可见文本为 `周四 09:18`,hover title 为 `2026-06-25 09:18:55 Asia/Shanghai`。
- 当前本地时间为 `2026-06-30 12:00:00`,记录本地时间为 `2026-06-02 09:18:55` ->
  最近记录可见文本为 `06-02 09:18`。
- 当前本地时间为 `2026-06-30 12:00:00`,记录本地时间为 `2025-12-30 09:18:55` ->
  最近记录可见文本为 `2025-12-30 09:18`。
- 操作员详情/skill 详情"最近记录"构造仅有 `day` 无 `first_seen` 的记录,且 `today=2026-06-30`:
  `day=2026-06-30` -> 显示 `今天`;`day=2026-06-29` -> 显示 `昨天`;
  `day=2026-06-25` -> 显示 `周四`;`day=2026-06-02` -> 显示 `06-02`;
  hover title 均保留原始 `day`。
