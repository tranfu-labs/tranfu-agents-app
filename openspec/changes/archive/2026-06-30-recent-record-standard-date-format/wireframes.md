# 线框：recent-record-standard-date-format

基线:

- `docs/wireframes/pages/skill-detail.md`
- `docs/wireframes/pages/operator-detail.md`

本变更不改变布局结构,只改变最近记录时间列的示例和注释规则。

## pages/skill-detail.md

```
┌─ ⑦ 最近记录 ─────────────────────────────────────────────────────────────────────────────────────────────────────────┐
├─ 表格 ───────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│ 时间                │ 调度员 │ 运行时 │ mode │ session                                                              │
├─  ───────────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│ 5分钟前             │ opA │ Claude │ used     │ s_abc123…                                                           │
│ 昨天 23:00          │ opB │ Codex  │ equipped │ s_def456…                                                           │
│ 周四 09:18          │ opC │ Hermes │ used     │ s_xyz789…                                                           │
│ 06-02 09:18         │ opD │ Codex  │ used     │ s_old222…                                                           │
└──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┘
hover(时间): 2026-06-27 23:00:00 Asia/Shanghai
```

注释 ⑦ 更新为:时间列取 `first_seen` 按浏览器本地时区展示;本地今天内显示相对时间,昨天显示 `昨天 HH:mm`,
2-6 天前显示星期标签加 `HH:mm`,今年更早显示 `MM-DD HH:mm`,跨年显示 `YYYY-MM-DD HH:mm`;hover 显示完整本地绝对时间到秒+时区。
缺失 `first_seen` 时按原始 `Asia/Shanghai` 统计日 `day` 与服务端 `today` 显示 `今天` / `昨天` / 星期 / `MM-DD` / `YYYY-MM-DD`,
hover 保留原始日期;手机为摘要行;不可点。
