# Agents 明细“距上次活跃”文案差异

基线：`docs/wireframes/pages/agents.md`。

## pages/agents.md

版式、列宽、断点和交互不变，只明确第 ⑥ 区最右列是“当前时间距最后一次活跃”的相对时间，不是运行时长。

### 桌面差异

```text
┌─ ⑥ // Agent 明细 ────────────────────────────────────────────────────────────────────────────────────────────────────┐
│Agent / 状态 / 任务             │操作员│今天时长/天数│累计质量│Skills/MCP│Shim   │距上次活跃              │
│build · 运行中 › 构建 dashboard │alice │4h32m / 1天   │91%     │3 / 3     │current│9秒前 ›                 │
└──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┘
```

### 手机差异

```text
┌─ 明细表摘要行 ⑥ ────────────┐
│build · 运行中           ›   │
│操作员 alice                 │
│今天 4h32m / 1天 · 累计91%   │
│Skill3 · MCP3 · current      │
│距上次活跃 9秒前             │
└─────────────────────────────┘
```

英文对应标题为 `Time since last active`。
