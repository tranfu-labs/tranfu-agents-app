## 注释

| 编号 | 元素 | 状态 / 交互 | 数据来源 | 引用控件 |
|---|---|---|---|---|
| ① | 控制条 / 手机摘要 | 固定 Agent × 运行时长口径；保留搜索、状态、时间、排序；旧 `rank/rt/op` 清理 | `/agents` URL | Agents toolbar |
