### 平板与手机

平板沿真实页顺序单列 skeleton；手机沿真实 DOM 顺序显示“控制摘要 → 问题线索 → Agent 明细 → KPI → 排行 → 趋势”的 skeleton。所有占位块宽度受页面根约束，长趋势占位只在图表盒内，不产生根横向滚动。

custom 起止时间尚未填完整时，① 使用真实可交互控制条并继续显示两个日期输入；②–⑤ 数据区保持 skeleton。完整 query 切换同样保留真实控制条、只替换数据区，避免旧排行榜与新 URL 同屏。
