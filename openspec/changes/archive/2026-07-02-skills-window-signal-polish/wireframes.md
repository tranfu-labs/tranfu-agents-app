## 注释 delta

| 编号 | 修改 |
|---|---|
| ① | 控制条删除环比开关;`cmp` 只作为旧 URL no-op 保留。 |
| ② | 区块标题从「过去 W 变化」改为当前时间窗变化 label;所有句内 `W` 改为时间窗 label 派生。 |
| ③ | 问题线索不显示具体 skill/operator 例子;保留数值、行动结论和 icon 证据入口。 |
| ④ | 按人视角的操作员排行按 `sessions_window` 排序,继承 `runtime/source`,不继承 skill 搜索、Top N、隐藏 0 使用。 |
