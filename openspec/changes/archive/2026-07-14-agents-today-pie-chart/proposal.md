# 提案：agents-today-pie-chart

## 背景

Agents 列表默认展示“今天”，当前趋势面板把唯一统计日画成一根堆叠柱。单根柱能够表达总量，却不利于快速比较今天各操作员或运行终端的构成；同时它会让用户误把单日快照当成时间趋势。

## 提案

- 当 Agents 当前窗口只有一个真实统计日且当前指标为正值时，把趋势面板改为环形扇形图，按当前操作员或运行终端视角展示 Top 8 + 其他的占比。
- 环形扇形图复用现有柱状图的分组颜色、图例按钮、悬停降权、tooltip、指标切换、主题变量与键盘日期停靠点，保持同一套图表语言。
- 环中心展示当前指标总值；面板上方继续同时展示当日活跃 Agent 数与活跃时长。
- 两日及以上窗口继续使用现有逐日堆叠柱状图；当前指标全窗为零时继续显示 Empty。
- 新增纯函数计算单日扇区的值、占比与累计偏移，并补充单元测试与浏览器视觉验证。

## 影响

- 影响 `frontend/src/components/agents/AgentActivityChart.tsx`、图表辅助逻辑、`frontend/src/lib/agentsDashboard.ts`、对应单测与 `frontend/src/styles.css`。
- 更新 board 事实源、Agents 字符线框与项目操作手册中的单日图表规则。
- 不改变服务端 API、统计口径、URL、排行、多日图、事件协议或数据库结构。
