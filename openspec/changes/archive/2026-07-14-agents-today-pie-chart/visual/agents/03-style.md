# 三、风格 + 实践 — 智能体列表页单日活跃分布

## 1. 页面性质判定

- **位置**：实用。
- **理由**：主场景是快速比较今天各分组的构成并查看精确事实，因此优先延续现有仪表盘的数据密度、颜色与交互，不引入装饰性图形。

## 2. 组件清单

| 组件类型 | 出现于哪个区域 | 主要用途 | 变体 |
|---|---|---|---|
| 指标分段按钮 | 面板头部区 | 切换活跃智能体数与活跃时长 | 默认、选中、聚焦 |
| 单日摘要 | 单日摘要区 | 并列显示双总计 | 默认 |
| 扇形图 | 扇形图区 | 表达当前指标各分组占比 | 默认、指向、聚焦 |
| 中心数值 | 扇形图区 | 显示当前指标总值 | 人数、时长 |
| 图例按钮 | 图例区 | 映射分组名称与颜色并联动高亮 | 默认、指向、聚焦 |
| 明细浮层 | 明细浮层区 | 显示日期、分组值和双总计 | 左侧、右侧、视口避让 |
| 空状态 | 空状态区 | 当前指标全零时说明 | 无数据 |
| 堆叠柱状图 | 多日图区 | 保留多日逐日趋势 | 短窗口、长窗口横滚 |

自查：显示框架段中的所有区域与状态均有对应组件。

## 3. 调性候选

### 候选「同构数据环」

- **对标**：本项目现有 Agents 多日堆叠柱状图与图例。
- **色板**：完整复用现有分组颜色映射、正文色、弱文字色、边框色与页面背景变量。
- **字号比例**：10、11、12、13、16、20 像素。
- **留白密度**：中等偏紧，保持运营看板扫描效率。
- **特点**：使用有中心读数的环形扇形图；扇区描边、透明度、图例与浮层行为和柱状图同构。
- **文件**：`03-alt-data-ring.html`。

### 候选「紧凑实心扇区」

- **对标**：常见运营分析页的实心占比图，但仍复用本项目现有配色与图例。
- **色板**：与候选「同构数据环」一致。
- **字号比例**：10、11、12、13、16、20 像素。
- **留白密度**：紧。
- **特点**：使用实心扇形区，中心总值移到图下；面积感更强，但和现有轻量柱状图的视觉重量差距更大。
- **文件**：`03-alt-solid-pie.html`。

## 4. 选定 + 原因

- **选定**：候选「同构数据环」。
- **理由**：用户明确要求扇形图和现有柱状图尽可能统一。数据环能沿用同一分组色、透明度、线性描边、图例和浮层，同时通过中心总值强化单日快照，视觉重量也更接近现有细柱。
- **选定文件**：`03-selected.html`。
