# 一、显示信息 — Agents 列表页单日活跃分布

> 目的：锁定“今天”趋势图需要表达的信息，不改动 Agents 列表页其他区域的信息优先级。

## 1. 用户画像

- 团队负责人：快速判断今天的 Agent 活跃由哪些操作员或运行终端构成。
- 平台治理人员：在活跃 Agent 数与活跃时长之间切换，定位单日分布是否过度集中。

## 2. 用户场景

### 主场景

- **查看今天的构成**：进入默认“今天”窗口后，直接比较各操作员或运行终端的占比。
- **切换观察口径**：在活跃 Agent 数与活跃时长之间切换，观察同一批分组的构成变化。

### 次场景

- **查看精确明细**：通过悬停、点击或键盘聚焦查看各分组值、当日总人数和总时长。

### 反场景

- 不为单日伪造小时级时间序列。
- 不改变多日窗口的逐日趋势表达。
- 不改变排行、筛选、明细表或数据统计口径。

## 3. 浏览路线

- **查看今天的构成路线**：进入 Agents 列表页 → 保持“今天”窗口 → 查看扇形分布与图例 → 离开页面或下钻其他区域。
- **切换观察口径路线**：查看单日扇形分布 → 切换活跃 Agent 数或活跃时长 → 比较扇区占比 → 查看精确明细。
- **查看精确明细路线**：指向、点击或聚焦扇形 → 阅读浮层中的分组值与双总计 → 关闭浮层。

## 4. 字段清单 + 优先级

| 字段 | 数据源 | 量级 | 优先级 | 服务于哪些场景 |
|---|---|---|---|---|
| 当前指标各分组值 | 单日趋势模型的分组数组 | 最多九项 | 必看 | 查看今天的构成、切换观察口径 |
| 当前指标总值 | 单日活跃 Agent 数或活跃时长 | 一个 | 必看 | 查看今天的构成、切换观察口径 |
| 分组名称 | 操作员或运行终端视角 | 最多九项 | 必看 | 查看今天的构成、查看精确明细 |
| 分组颜色 | 现有图表颜色映射 | 最多九项 | 重要 | 查看今天的构成 |
| 当日活跃 Agent 总数 | 单日汇总 | 一个 | 重要 | 查看精确明细 |
| 当日活跃总时长 | 单日汇总 | 一个 | 重要 | 查看精确明细 |
| 统计日期 | 当前窗口唯一统计日 | 一个 | 辅助 | 查看精确明细 |
| 非当前指标的分组值 | 单日趋势模型 | 最多九项 | 留档 | 仅浮层辅助阅读 |

## 5. 对比/聚合单元

一条记录的粒度：

- 是：当前统计日下，一个操作员或运行终端分组在当前指标中的聚合值。
- 不是：小时级记录、单次会话或单个事件。
