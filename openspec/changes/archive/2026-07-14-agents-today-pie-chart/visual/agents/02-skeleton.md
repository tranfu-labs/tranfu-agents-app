# 二、显示框架 — Agents 列表页单日活跃分布

> 目的：把单日字段落到趋势面板内的固定区域与状态，不讨论颜色、圆角等视觉细节。

## 1. 区域划分

| 区域名 | 容纳字段 | 主要服务场景 |
|---|---|---|
| 面板头部区 | 当前视角、活跃 Agent 数与活跃时长指标切换 | 切换观察口径 |
| 单日摘要区 | 当日活跃 Agent 总数、当日活跃总时长 | 查看今天的构成、查看精确明细 |
| 扇形图区 | 当前指标各分组值、当前指标总值、分组颜色 | 查看今天的构成、切换观察口径 |
| 图例区 | 分组名称、分组颜色 | 查看今天的构成 |
| 明细浮层区 | 统计日期、各分组当前指标值、当日双总计 | 查看精确明细 |
| 空状态区 | 当前指标无正值时的说明 | 查看今天的构成 |
| 多日图区 | 多日逐日堆叠柱与现有交互 | 多日窗口观察，不属于单日扇形变化 |

对应关系自查：显示信息段中的必看与重要字段均已落入上述区域。

## 2. 状态机

- **单日入口态**：面板头部区显示视角与指标切换；单日摘要区显示双总计；扇形图区显示分组占比和中心总值；图例区常驻；明细浮层区隐藏。
- **单日指向态**：单日入口态保持；当前分组扇区与对应图例保持高权重，其余扇区降权；明细浮层区显示分组明细和双总计。
- **单日键盘态**：图表只有一个日期停靠点；聚焦后显示与指向态相同的浮层，方向键不制造额外日期，退出键关闭浮层。
- **单日空数据态**：当前指标全日为零时只显示空状态区，不显示扇形、坐标轴或日期。
- **多日态**：当窗口包含两个及以上真实统计日时显示现有多日图区，不显示单日扇形图区。

## 3. 区域间关系

- 面板头部区切换指标 → 单日摘要区不变 → 扇形图区按新指标重新计算占比与中心总值。
- 控制条切换操作员或运行终端视角 → 扇形图区与图例区同步更换分组。
- 指向扇区或图例 → 对应扇区与图例联动高亮，其他扇区降权。
- 点击或聚焦扇形图区 → 明细浮层区显示；退出键、移出面板、滚动或点击图外关闭。
- 时间窗从今天切到多日 → 单日摘要区与扇形图区退出，多日图区进入；反向切换同理。

## 4. 浏览路线落到区域

- **查看今天的构成路线**：进入单日入口态 → 扇形图区比较面积 → 图例区确认分组名称 → 退出。
- **切换观察口径路线**：单日入口态 → 面板头部区切换指标 → 扇形图区更新 → 比较新占比。
- **查看精确明细路线**：单日入口态 → 指向、点击或聚焦扇形图区 → 单日指向态或键盘态 → 阅读明细浮层区 → 关闭。
